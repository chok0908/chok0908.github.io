2. Open Map

선택한 Map을 새 Screen 창으로 열어주는 기능이다.

- Rename 버튼을 클릭하면 선택된 Map의 기존 이름이 선택된 상태로 Text InputBox가 생성되어 바로 새 이름을 입력할 수 있게 된다.

- 다중 선택된 상태에서는 버튼은 비활성화 상태가 된다.

- 이름 적용은 Enter 키 또는 Focus Out이 되면 적용된다.

- Map Rename의 경우 이름 수정이 완료된 시점 부터 Map이름과 관련된 곳에 적용된다.
