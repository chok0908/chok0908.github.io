마우스 오버 영역(1px)

- Full Screen를 클릭하면 현재 선택된 Screen이 모니터 화면에 꽉 채운다.

- Full Screen 된 Screen은 항상 최상위에 위치한다.

- Full Screen은 Screen 단위로 된다.

- Screen이 모니터 화면에 꽉 차게 사이즈 변경이 되면서 안내 문구가 3초 동안 보여주고 사라진다.

- ESC 키를 누르거나 상단바에 Exit 버튼을 클릭하면 Screen은 이전 상태 크기로 돌아간다.

- 상단 1px에 마우스 오버하면 상단바가 나온다.

- 상단바에는 타이틀과 Exit 버튼이 있다.

- 상단바 영역에서 마우스 커서가 아웃되면 상단바는 자동으로 사라진다.

- 각각 모니터마다 Float 또는 Dock 상태와 상관없이 마지막에 선택된 Screen이 Full Screen 된다.

- 자신이 속해 있는 모니터 화면에 Screen이 Full Screen 된다.

- Float 된 Screen은 여러 모니터에 속해 있을 수 있다. 그럴 경우에는 제일 많은 영역에 속해 있는 모니터 화면에 Full Screen 된다.
