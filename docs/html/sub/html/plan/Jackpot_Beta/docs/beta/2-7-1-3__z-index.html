2. Toolbar의 Z-index기능 실행

Objects패널의 순서에 의한 Z-index

- Objects Panel의 순서는 Map상에서의 Object의 Z-index를 의미한다.

- Objects Panel에서 최상단에 있는 Object가 Z-index상 최상위에 위치한다.

- Objects Panel에서 순서가 바뀌면 Z-index값도 바뀐다.
