9. Multi Select

< Multi Select 방법 >

- 영역을  Mouse Drag

- 키보드의 Shift를 누른 상태에서 Object를 선택 ( 직교 기능 포함 )

- 키보드의 Ctrl을 누른 상태에서 Object를 선택 ( Copy 기능 포함 )


< Multi Select 이후 행위 > 

- 선택된 Object들의 공통속성을 활성화 시켜줌

- 공통속성을 사용자가 바꾸어주면 Group안에 존재하는 모든 Object들은 같은 속성값으로 가지게 됨
