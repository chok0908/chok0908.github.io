- 일기장 / 조성찬 -

싫었던 기억... 아팠던 기억.. 슬펐던 기억.
지우고 싶은 지난 나의 기억들
아무리 지우고 지워봐도
깨끗하게 지워지지 않는다
왜 이리 꾹꾹 눌러 썼을까
늘어나는 지우개 가루와
눌려진 종이 자국이
내 지난 과거를 더욱더 선명하게 만든다

" 지난 과거를 지우려 노력하기 보다. 그때의 나를 인정해 주자. "
