2-7-2-2. Rename

1. Rename

- Object를 선택하고 Rename을 실행하게 되면 Object의 기존이름이 전체 선택된 상태의  Text InputBox가 생성되어 이름을 입력할 수 있게 한다.

* Multi Select가 된 상태에서는 Rename기능을 사용할 수 없다.

* Multi Select상태에서는 Rename기능만 비활성화 처리된다.

* Object가 선택되지 않는 상태에서도 비활성화 처리된다.

* 1개의 Object가 선택된 상태에서만 활성화 된다.
