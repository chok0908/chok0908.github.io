최초 생성 이후 새 Map을 만드는 경우

- New를 클릭하면 새 Screen Tab을 생성하여 새 Map을 만든다.

- 새 Map 이름은 자동생성되어 나온다.

- 새 Map은 사용자가 저장하기 전까지는 임시저장되어 보여진다.

- 새 Map은 항상 맨 왼쪽에서 만들어져 Tab이 선택되어 나온다.

- 새 Map이 만들어질 때 기본 Map들이 있는 경우 새 Map 오른쪽으로 모두 밀린다.
