다른 이름 저장 팝업창

- Save As를 클릭하면 팝업창이 나와 다른 이름으로 저장한다.

- Map name 입력창에 Map 이름이 자동으로 입력되어 모두 선택된 상태로 나온다.

- 이름을 변경하고 저장하면 다른 이름으로 Map이 저장된다.

- 이름을 변경하지 않고 그대로 저장할 경우 확인 팝업창이 나온다.

- 예 버튼을 클릭하면 현재 선택된 Screen에 덮어쓰기되며 모든 팝업창이 사라진다.

- 아니요 버튼을 클릭하면 확인 팝업창만 사라진다.
