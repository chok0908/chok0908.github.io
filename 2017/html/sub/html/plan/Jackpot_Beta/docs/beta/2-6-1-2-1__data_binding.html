- Data Binding을 버튼을 누르게 되면 Local PC와 Server를 선택하게 된다.

- Server의 경우 Jackpot서버를 통해 연결된 Data 서버를 선택 할 수 있다.


 < Data 로컬 경로 지정 >

- Local PC를 선택하게 되면 Local경로를 선택할 수 있는 Popup창이 뜨게 된다.

 ( Popup창의 경우 .csv 확장자 필터링 되어 보여진다. )

- .csv 확장자가 아닌 다른 파일을 선택할 경우 아무런 동작없이 창이 닫히게 된다.

 ( 로컬 Data 선택하는 부분은 초기화 된다. )

- Default 폴더 위치는 Engraver가 설치된 파일 위치가 열린다.

- 사용자가 파일을 가져온 이력이 있는 경우 최근 파일을 가져온 경로로 파일 위치가 열린다.


* 가져온 Data는 Map이 저장되는 시점에 서버에 저장이 된다. (저장 위치는 개발에서 결정함)

* Server의 경우 Beta Version이후에 기획
