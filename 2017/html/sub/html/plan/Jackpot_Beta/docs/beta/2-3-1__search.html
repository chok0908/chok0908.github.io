2. Tag 검색

"#"을 맨 앞에 쓰고 띄지 않고 검색어를 입력하면 Tag 검색이 된다.

- 모든 문자를 검색어로 사용할 수 있다.

- Tag 검색에 해당되는 않는 경우 "#"은 그냥 문자로 인식하여 검색한다.

  map#

- Tag 검색어는 하나만 사용할 수 있다.

- 맨 앞에 "#"만 Tag 문법으로 인식하고 그 이외는 모두 문자로 인식하여 검색한다.

  #factory, #korea

"#" 문자 입력

Tag 문법 이후 문자 입력
