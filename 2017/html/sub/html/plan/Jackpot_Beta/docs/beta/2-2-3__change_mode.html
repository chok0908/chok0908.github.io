EDIT

Change Mode

- Change Mode 버튼을 클릭할 때 마다 Edit / Play Mode가 변경된다.

- Toggle 버튼이다.

- Toggle이 On 되면 Edit Mode이고 Off 되면 Play Mode가 된다.

- Edit Mode 일 때는 Map 편집이 가능하다.

- Play Mode 일 때는 Map 보기만 가능하다.

- 선택된 Screen에만 Mode 변경이 적용된다.

       Map

- Play Mode가 되면 Screen Tab 타이틀에 아이콘 표시가 생긴다.

< 비활성화 >

- Properties Panel에 타이틀바 영역을 제외하고는 비활성 된다.

- Libraries Panel에 타이틀바 영역을 제외하고는 비활성 된다.

- Objects Panel에 하단 버튼 영역이 비활성화 된다.


< 활성화 >

- Maps Panel은 활성화 되어 있다.

- Maps에서 사용되는 모든 기능이 가능하다.

- Objects에 검색과 리스트 영역을 활성화 시켜 Play Mode에서 해당 Object로 이동할 수 있게한다.

- Objects에서 Object를 선택하는 기능 이외에 다른 기능을 막는다.

- Objects의 Context Menu도 지원하지 않는다.

- Map을 Open한 경우에는 Change Mode의 Default 값 Play Mode이다.
