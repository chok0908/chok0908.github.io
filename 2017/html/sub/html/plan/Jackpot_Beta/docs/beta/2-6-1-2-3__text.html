- Content : Text 내용을 입력할 수 있다.

  ( Content의 입력내용은 Screen의 Map에 올려진 Object와 실시간 연동 / < Default값 - 없음 > )




 - Color : Text의 색상을 설정한다. ( Color Picker / < Default값 - #ffffff > )

 - Family : Text 글꼴을 설정한다.  ( Window Font적용 / < Default값 - Arial > )



 - Size : Text 크기를 설정한다.  ( Pixel단위 / < Default값 - 12px > )




 - Style : Text Style 설정한다.

  ( 체크박스로 동작 / Bold : 글자두께 두껍게 / Italic : 글자모양 기울이기 / Under Line : 글자에 밑줄포함 / < Default값 - 없음 > )

 - Align : Text 정렬 방식을 설정한다.  ( 토글 버튼으로 동작 / Horizontal < Default값 - Center > / Vertical < Default값 - Center > )

 - Line Spacing : Text의 줄 간격 설정한다.  ( Pixel단위 / < Default값 - Auto >  )




 - Padding : Text의 여백 설정한다.  ( Pixel단위 / < Default값 - 2 >  )
