2-1-3-2. Fit All

Object 모두 보기

- View > Fit All을 클릭하면 현재 Zoom Level에서 Object들이 모두 보이는 Zoom Level로 변한다.

- Screen을 중심을 기준으로 Zoom Level이 변한다.

- Screen 위치는 전체 Object가 보이는 영역이 Screen 중심에 온다.

- Fit 될 때는 Screen을 기준으로 Fit을 100%로 볼 때 80%정도 보일 정도로 Zoom Level이 되어야 한다.

전체 Object가 보이는 영역

Zoom Level 변경 정의

- 전체 Object가 보이는 영역이 Screen 영역보다 클 경우 Zoom Level은 축소된다.

- Zoom Level이 축소되는 값은 Screen 영역에 전체 Object가 보이는 영역이 Fit 되는 정도가 된다.

Screen 영역

Screen 영역

- 전체 Object가 보이는 영역이 Screen 영역보다 작을 경우 Zoom Level은 확대된다.

- Zoom Level이 확대되는 값은 Screen 영역에 전체 Object가 보이는 영역이 Fit 되는 정도가 된다.

Fit All과 Fit Selection 차이점

Fit All은 Object 선택과 상관없이 모든 Object가 Fit 되게 보여주는 기능이고 Fit Selection은 사용자가 선택된 Object만 Fit 되게 보여주는 기능이다.
