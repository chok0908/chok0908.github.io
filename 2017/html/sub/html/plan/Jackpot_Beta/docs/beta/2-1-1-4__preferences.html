1. Interface

프로그램 인터페이스 테마 색상을 설정한다.

OK 버튼을 클릭하기 전 까지는 사용자가 설정한 값은 임시저장된다.

2. Units

수치의 기준이 되는 Zoom Level을 설정한다.

- Preferencest를 클릭하면 팝업창이 나온다.

- 테마 색상은 Dark, Gray, Light로 3가지 색상을 지원한다.

- Default 값은 Dark 이다.

- 색상을 선택하면 인터페이스 색상이 바로 적용된다.

- 기준 레벨을 선정하게 되면 모든 수치를 기준 레벨의 값으로 표시하게 된다.

- Default 값은 Level 10이다. ( Zoom Level은 1~20까지 주어진다. )

- 작업중에 기준레벨을 바꿀 경우 바뀐 기준레벨을 기준으로 다시 산정되어 계산된 크기값이 모든 Properties에 적용된다.
