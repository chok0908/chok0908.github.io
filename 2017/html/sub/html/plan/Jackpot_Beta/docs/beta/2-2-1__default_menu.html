Set Home 설정 후

- Set Home이 설정이 된 경우에만 버튼이 활성화된다.

Edit or Play

Home 실행

- 편집 여부와 상관 없이 Home 명령은 실행된다.

- Object가 선택된 상태에서 Home을 실행해도 선택된 Gripper는 유지된다.

- Home은 위치와 Zoom Level 이외에는 아무런 영향을 주지 않는다.

- Default 값은 On 상태이다.

- On 상태가 되면 마우스 커서 모양이 화살표 모양이 된다.

- Pan 또는 Zoom 버튼을 클릭하면 Off 상태가 된다.

- Default 값은 Off 상태이다.

- Pan 버튼을 클릭하면 On 상태가 된다.

- On 상태가 되면 마우스 커서 모양이 손바닥 모양이 된다.

- Selection 또는 Zoom 버튼을 클릭하면 Off 상태가 된다.

- 마우스 휠을 통해 Map 확대/축소 된다.

- 마우스 드레그로 Map 이동된다.

- Default 값은 Off 상태이다.

- Zoom 버튼을 클릭하면 On 상태가 된다.

- On 상태가 되면 마우스 커서 모양이 돋보기 모양이 된다.

- Selection 또는 Pan 버튼을 클릭하면 Off 상태가 된다.

- Object를 마우스 클릭하면 Object가 선택이 된다.

- 마우스 커서가 Map 위에 있을 때 마우스 휠을 사용하면 커서 좌표를 중심으로 Map이 확대/축소 된다.

- 마우스 휠을 아래로 하면 확대되고 위로 하면 축소된다.

- Map에서 마우스 드레그하면 드레그 방향으로 Map이 이동 한다.

- Pan이 On 상태가 되면 Object 위에서 마우스 드레그가 가능하여 Map을 이동 시킬 수 있게 된다.

- Zoom이 On이 되면 마우스 클릭으로 Map을 확대/축소할 수 있다.

- Zoom이 On이면 Default 값은 확대이다.

- Map을 축소하려면 Alt 키를 누른 상태로 마우스 클릭하면된다.

- Alt 키를 누른 상태면 아이콘이 해당 축소 아이콘으로 변한다.

- Alt 키를 때면 다시 확대 아이콘으로 변한다.

Alt 키를 누른 상태 유지

Map을 설정된 홈 화면으로 돌아가기

Map 컨트롤 하기

- Space키를 누르는 순간만 Pan 기능을 사용할 수 있다.

- Selection 또는 Zoom을 사용할 때에도 사용가능 하지만 Space키를 누른 상태에서만 적용된다.

- Space키를 누르면 Pan 기능을 되지만 Pan이 On 상태 표시로 바뀌는 것은 아니다.(Selection 또는 Zoom이 On 상태가 유지되어야 함)
