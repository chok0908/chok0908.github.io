1. Group으로 설정된 Object편집

Map_01

Group설정된 Object를 편집할 수 있는 경우


1. Objects Panel에서 Group으로 묶인 Object를 선택할 경우

2. Screen에서 Group을 선택한 뒤 Group안의 Object를 선택할 경우

( 처음엔 Group이 선택되고 이후 Group안의 Object가 선택된다.)


Group내 Object를 편집 할 수 있다.


* Group 영역 외의 공간은 짙은 회색( #333333 )의 배경이 Opacity - 10%가 된 상태에로 덮힌다.

* Group 영역만 포커싱된다.

* Group 영역 안의 다른 Object를 선택하게 되면 Object가 선택된다.

* Group 영역 밖의 다른 Object를 선택할 경우 짙은 회색 배경이 사라지면서 Object가 선택된다.
