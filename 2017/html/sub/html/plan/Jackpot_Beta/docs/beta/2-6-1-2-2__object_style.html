< Object Type >

 - Object Type을 통해 Object의 속성이 바뀐다.

 - Object Type의 종류에는 Line / Triangle / Rectangle / Pantagon / Hexagon / Circle / Star / Grid / Chart / Table가 있다.

( 하나의 Object는 한가지의 Type을 가지게 된다. )

 - Line : 선의 형태를 가진 Object Type이다.

 - Triangle : 삼각형의 도형 가진 Object Type이다.

 - Rectangle : 사각형의 도형 가진 Object Type이다.

 - Pantagon : 오각형의 도형 가진 Object Type이다.

 - Hexagon : 육각형의 도형 가진 Object Type이다.

 - Ellipse : 타원형의 도형 가진 Object Type이다.

 - Star : 별 모양의 도형 가진 Object Type이다.

 - Chart : 연결된 Data를 Chart형태로 표출할 수 있다. ( Bar / Pie / Line / ... )

 - Table : 연결된 Data를 Table 형식의 표의 형태를 가지고 있다.
