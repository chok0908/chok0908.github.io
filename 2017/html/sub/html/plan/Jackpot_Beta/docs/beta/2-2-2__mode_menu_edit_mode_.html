3. Cut

선택된 Object를 잘라낸다.

작업이 있을 때

- Default 값은 비활성화 상태이다.

- 사용자가 Map에서 작업을 하는 시점에서 활성화 된다.

- 최대 20개 이상 실행시키면 비활성화 상태가 된다.

Undo 한 후

- Default 값은 비활성화 상태이다.

- 사용자가 Undo 하는 시점에서 활성화 된다.

- Undo 개수 이상 실행시키면 비활성화 상태가 된다.

Object 선택 후

- Default 값은 비활성화 상태이다.

- 사용자가 Object를 선택한 시점에서 활성화 된다.

Crtl + X

- Object를 잘라내기 하면 선택된 Object는 사라지고 Object가 복사된다.

4. Copy

선택된 Object를 복사한다.

Object 선택 후

- Default 값은 비활성화 상태이다.

- 사용자가 Object를 선택한 시점에서 활성화 된다.

Ctrl + C

- Object를 복사 하면 선택된 Object는 그대로 있고 Object가 복사된다.

- 사용자가 Alt키를 누른 상태에서 마우스 드레그 하면 드레그한 위치에 Object가 복사된다.

- 사용자가 Alt키를 누른 상태에서 선택된 Object 영역내에 마우스 오버하면 마우스 커서 모양이 바뀐다.
