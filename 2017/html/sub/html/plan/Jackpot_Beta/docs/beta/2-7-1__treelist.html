< Map Name >

-  Screen의 Tab에 Assign된 Map Name이 표시된다.

-  List에서 Map Name은 수정할 수 없다.

-  List에서 Map Name을 선택하면 Map이 선택되면서 Map Properties가 제공된다.


< Group Name >

- Group Name은 Object List에서 수정이 가능하다.

- Group Name이 있는 경우 Group Name이 표시되고 없는 경우 Group이라고 표시 된다.

- Properties의 Group Name과 실시간 연동된다.


< Object Name >

- Objcet Name은 Object List에서 수정이 가능하다.

- Object Name이 있는 경우 Object Name이 표시되고 없는 경우 Object Type이 이름이 표시된다.

- Properties의 Object Name과 실시간 연동된다.


< Object Lock & Unlock >

- Object를 Map상에서 편집모드를 잠그는 기능이다. 

  (  Default값 - 잠금 해제  )



< Object Show & Hide >

- Object가 Map Editing 상황에서의 Visibility를 설정하는 기능이다.

 ( Default값 - View )



- Multi Select 기능

   Object는 다중 선택이 가능하다.

   Grouping, Duplicate, Delete 기능 동작

2. Object Name

< Object name 이 없는 경우 >

- Object에 Name을 지정해 주지 않으면 Object Type이 Object Name란에 워터마크 처리되어 나온다.

- Group에 Name을 지정해 주지 않으면 "Group"이라고 Object Name란에 워터마크 처리되어 나온다.

- 이름이 없는 Object 나 Group을 Copy하면 Object Name란에 ( Object Type / Group )워터마크 처리되어 나온다.

 ( '복사 대상 이름 + _ + Copy'가 생기지 않음 )

- Object Name은 동일이름을 허용한다.

- Group Name도 동일이름을 허용한다.



< Object name이 있는 경우 Copy >

- 최초에는 '복사 대상 이름 + _ + Copy'로 자동생성 된다.

- 최초 복사한 이후 동일한 것을 또 복사한 경우 '복사 대상 이름 + _ + Copy + 빈칸 + 자동생성 숫자'로 자동생성 된다.
