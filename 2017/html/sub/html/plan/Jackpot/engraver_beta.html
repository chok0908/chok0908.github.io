버튼을 클릭하면 해당 기획서 페이지로 이동합니다.

JP

     쉽고 편리하게 다양한 데이터를 이용하여 Map을 만들 수 있는 MODE

      동적 데이터 / CSV 데이터 / Office 문서 파일 / Camera 등... 다양한 형태의 파일들을 불러와 Map을 작성할 수 있다.

VIEWER

     MAP으로 만들어진 파일들을 볼 수 있는 MODE

      Map List를 통해 작성된 Map들을 불러와 한눈에 확인이 가능하다.

 SETTING

     외부 데이터를 연결하고 프로그램의 설정부분을 편집 할 수 있는 MODE

      동적 / 정적 / Camera 등.. 외부 Data와 Engraver를 연결 할 수 있다.
