< Align > - 선택된 객체들을 정렬시켜준다.

          Left - 왼쪽으로 정렬

          Center - 중앙으로 정렬

          Right - 오른쪽으로 정렬

          Top - 위로 정렬

          Middle - 중간으로 정렬

          Bottom - 아래로 정렬


 < Distribute > - 선택된 객체들을 균일한 간격으로 배치한다.

          Horizontal - 가로 방향으로 균등히 배치

          Vertical - 세로 방향으로 균등히 배치

 

 < Visibility >

          Lock to Screen - 스크린에 객체를 고정

          Keep Size - Zoomable과 관계없이 Map상에서 Size를 고정

          Fit - 선택된 객체를 스크린 전체 사이즈로 맞춤

          Hide - 객체의 기본 설정값을 숨김으로 설정
