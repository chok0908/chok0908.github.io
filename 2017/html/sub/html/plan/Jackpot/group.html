1. Group

 - Group으로 된 묶어준 Object를 선택한 경우 Group에 대한 Inspector를 제공


< Size >

 - Map상에 X값과 Y값을 입력하여 좌표를 설정 할 수 있음

 - 좌표의 0,0은 좌측상단 끝 모서리  * 하단 Screen_3내용 참조

 - Width 와 Hight를 선택하여 Size를 사용자가 정의 할 수 있음


< Visibility >

 - Lock to Screen :  Screen에서 Zoom, Panning 잠금기능

 - Fit : Map Size에 맞게 Stretch 시키는 기능

 - Hide : Map상에 Object를 숨기는 기능
