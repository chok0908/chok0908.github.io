1. 어떤 Map의 Inspector 인지 표시

4. Style

1) 콤보박스를 클릭하면 해당 팝업창이 아래와 같이 나온다.

2) Shape  선택에 따른 Disable 처리 항목들

* Shape 모양이 선인 경우에만 활성화 되어 나온다.

3. Properties

Inspector : Untitled

타이틀바 영역에 Inspector 옆에 선택된 Map 이름이 표시된다.

Map 이름이 없는 경우에는 Untitled로 표시된다.

현재 어떤 Map에 Object을 수정하는지 사용자가 인지할 수 있게 해준다.

2. Name

Inspector : A_Map01

  (Object Name)

사용자가 직접 입력하기 전까지는 Default 이름이 나온다.

Default 이름은 Map Name, Object Name, Group Name 이 있다.

Default 이름은 Null이므로 사용자가 입력하기 전까지는 이름이 없는 상태이다.
