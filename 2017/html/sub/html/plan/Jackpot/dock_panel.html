4. Dock Pannel 기능 설명

-  Floating된 Panel을 다른 Panel 제목표시줄에 드레그 하면 Tab형태로 바뀐다.

- Docking상태의 Panel아래에 Floating Panel을 마우스 드레그 하면 해당 Docking Panel아래로 Docking된다.

- Floating Panel은 Snap기능을 제공한다.


2. Docking / Foating

 Panel들의 제목표시줄을 드레그 하면 Floating상태로 전환됨

Floating 상태에서는 Panel 상단에 닫기 버튼이 생성됨

Panel Docking상태에서는 Panel의 닫기 버튼을 제공하지 않음

3. Screen과 Panel

- Panel 영역과 Screen 영역은 서로 공유될 수 없다.

- Screen영역을 기준으로 Panel은 상하좌우 바깥쪽으로 붙을 수 있다.

- Docking과 Foating하는 순간 Screen영역의 Ratio는 변하지 않는다.
