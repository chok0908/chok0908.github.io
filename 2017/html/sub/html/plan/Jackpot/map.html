1. Map_ Advanced

- Map에는 Style이 존재하지 않음  :  Style 비활성처리

< Size >

 - Width 와 Hight를 선택하여 Size를 사용자가 정의 할 수 있음


< Tag >

 - Tag를 입력하면 하단에 Tag가 추가

 - Tag 삭제를 원하면 Tag목록에서 X버튼을 눌러 삭제

 

< Zoom >

- On : Zoomable이 가능한 Map으로 설정

- Off : Zoomabel이 불가능한 Map으로 설정
