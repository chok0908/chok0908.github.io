: 새로운 Map을 생성

: Object를 삭제

: Object를 좌측 기준으로 정렬

: Object를 우측 기준으로 정렬

: Object를 상단 기준으로 정렬

: Object를 세로 가운데 기준으로 정렬

: Object를 하단 기준으로 정렬

: Object들의 간격을 가로 방향으로 균일한게 배치

: Object들의 간격을 세로 방향으로 균일한게 배치

: 선택한 Object들을 하나로 묶어준다.

: Group을 해제한다.(각각의 객체로 분리)
