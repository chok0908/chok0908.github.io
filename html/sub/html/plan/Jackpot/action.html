< Action  List >

- Action List 목록에서 선택한 Trigger에 해당하는 설정 내용이 제공

- Target Object의 경우 Bring to Front, Show , Hide , Blinking , Invert , Text 은 복수 설정 가능  

- 그외  Open , Link, Go to Object , Change Map은 단일개의 선택만 가능

- Duration의 기본 단위는 0.1초


- 추가된 Action들의 내용이 List로 보여줌

- Action의 이름과 적용된 설정내용이 간략하게 표출됨

< Action이 적용된 Objcet >

- Action이 적용된 Objcet에는 Aciton ICON이 표시됨

10. Effect

< Invert Action >

- 해당 Object의 속성을 바꾸어주는 Action

 ex) hide  >  visibility    /   visibility > hide


< 공통 >

- Fade : 서서히 나타나고 사라지는 효과

- Slide : 미끄러지듯이 나타나는 효과


< Blinking >

- Scale : 객체가 커졌다가 작아졌다가 반복하는 효과

- Spread : 객체를 중심으로 안쪽에서 바깥쪽으로 파장이 반복적으로 퍼지는 효과
