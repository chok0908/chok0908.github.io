3. 이름 수정

선택된 상태에서 다시 한번 클릭하면 이름 수정이 가능하다.

이름 수정을 하면 Object 이름이 수정이 되는 것이다.

4. 선택

Object 목록에서 선택하면 Map에서 선택하는 것과 동일하게 Object가 선택된다.

다중 선택도 가능하다.

5. Z-Index

목록의 순서가 Map에 Z-Index 순서와 동일하다.

맨 처음에 있는 Object가 Map에서는 맨 위에 있는 것이다.

목록에서 순서를 변경하면 Map에서의 Object의 Z-Index가 변경되어야 한다.

목록 순서 변경은 드래그로 한다.

1. 어떤 Map의 Inspector 인지 표시

타이틀바 영역에 Object 옆에 선택된 Map 이름이 표시된다.

Map 이름이 없는 경우에는 Untitled로 표시된다.

현재 어떤 Map에 Object 목록인지 사용자가 인지할 수 있게 해준다.

6. 검색

검색 버튼 클릭하면 검색창이 나온다.

검색 버튼은 토글 버튼으로 다시한번 클릭하면 검색창이 사라진다.
