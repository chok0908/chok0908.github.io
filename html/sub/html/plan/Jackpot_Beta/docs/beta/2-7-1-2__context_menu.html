- Group : Object들을 Group 단위로 묶어주는 기능이다.

- Ungroup: Group 단위로 묵여진 Object를 개별 단위로 해제하는 기능이다.

- Rename : Object name 수정하는 기능이다.

- Duplicate : Object를 사본을 만드는 기능이다.

- Delete : Object를 삭제하는 기능이다.

- Objects 패널에서 Object 항목을 Mouse 우클릭을 하게 되면 Context Menu가 제공된다.
