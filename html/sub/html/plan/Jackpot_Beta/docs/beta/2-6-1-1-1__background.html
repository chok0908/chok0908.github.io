Image 로컬 경로 지정

 < Image 이름 >

- Image가 등록되지 않은 경우 아무런 표시가 없다.

- Image가 등록되는 순간 이미지의 이름이 표시되어진다.

- 이름영역을 벗어나는 장문의 이름의 경우 벗어나게 되는 영역으로 부터 [ ... ]표시를 해준다.


 < Image 로컬 경로 지정 >

- Import 버튼을 누르면 Image 선택 Popup이 Local PC의 경로가 열리게 된다

- Default 폴더 위치는 Engraver가 설치된 파일 위치가 열린다.

- 이미지가 아닌 다른 파일을 선택할 경우 아무런 동작없이 창이 닫히게 된다.

 ( 로컬 이미지 선택하는 부분도 초기화 된다. )

- Image 선택 Popup은 JPG / PNG / BMP / GIF / SVG형식의 파일만 볼 수 있게 Image Format을 분류해 준다.

- Image를 선택하면 Background에 Image가 배경으로 적용된다.

- 사용자가 Image파일을 가져온 이력이 있는 경우 최근에 파일을 가져온 경로로 파일 위치가 열린다.


* JPG / PNG / BMP / GIF / SVG형식의 이미지 파일을 읽어 올 수 있다.

* 가져온 Image는 Map이 저장되는 시점에 서버에 함께 저장이 된다.

* Image 이름이 Name영역을 벗어날 경우 - 벗어나는 Text를 [ ... ] 처리한다.

Map_01

- GIS를 선택하면 Map의 Background에 GIS Map이 입혀지게 된다.

- 검색 List의 목록을 선택하면 Screen에 보이는 Map 영역이 해당 지역을 중심으로 Zoom 되어 나온다.

(GIS Map 이 보여지는 모습은 GIS Map이 가지고 있는 자체 룰을 따른다.)

* GIS Map에 따라 검색기능 제공여부가 달라진다.

< Search bar >

- Search Bar에서 검색어를 입력한 후 Search버튼을 누르게 되면 검색이 되어진다.

( 키보드의 Enter키를 누르게 되어도 검색된다. )

- Map의 종류에 따라 검색창의 모양이 달라질 수 있다.

< List Reset >

- Clean Button을 누르게 되면 Search List의 항목이 Reset된다.

( Search List 목록이 모두 사라지게 된다. )

< Search List >

- 검색어에 해당하는 목록을 List로 뿌려준다.

< GIS Map 종류 >

- Preferences를 통해 등록된 GIS Map 종류가 Combobox형태로 나타난다.

- GIS Map이 없을 경우 GIS Map종류는 None으로 표시된다.

- GIS Map종류의 List는 이름 순서대로 나오게 된다.

- OpenStreet는 Default로 제공되는 GIS Map이며 검색기능은 제공되지 않는다.

 - Clean Button을 누르게 되면 Search bar의 내용과 Search Result List는 초기화 된다.
