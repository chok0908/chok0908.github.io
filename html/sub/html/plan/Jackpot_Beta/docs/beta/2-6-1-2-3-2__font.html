새로운 폰트로 대체할 경우

- Ok / Cancel 버튼을 누르면 해당 폰트가 Arial 폰트로 Screen에서 보여진다.

- Font Family의 속성은 변하지 않는다.

- 해당 Open Sans Font를 Local Pc에 설치하게 되면 Open Sans로 Screen에 나오게 된다.

- Ok 버튼을 누르면 Open Sans에 해당하는  폰트가 굴림폰트로 바뀌어 저장된다.

- Cancel 버튼을 누르면 해당 폰트가 Arial 폰트로 Screen에서 보여진다.

( Font Family는 ( Open Sans )로 저장되며 Screen에서는 Arial 폰트로 보여진다. )
