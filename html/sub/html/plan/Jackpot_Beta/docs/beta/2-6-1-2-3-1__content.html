Text Test

 - Content에 Text를 입력하게 되면 Screen의 Object에 실시간으로 등록된다.

 - Screen의 Object를 더블 클릭하여 Text를 입력하게 되면 Properties_Text_Content에 입력값이 실시간으로 등록된다.

 - Content영역에 있는 Text를 선택하여 Font 속성값을 바꿀수 없다. ( Color / Family / Weight / Size / Style / Align )

 - Content영역의 Text는 System Font로 나온다.

 - Font의 Color / Family / Weight / Size / Style / Align 기능을 적용하려면 Screen에 있는 Object를 선택하여 편집해야 한다.

Enter

- Object에 Data Binding이 된 경우 Data ID Button이 활성화 된다.

- Data ID Button을 누르게 되면 Data ID Popup창이 뜬다. ( Table 형태의 Data로 표출 )

- Data ID Popup이 화면상에 열려 있는 경우 Text만 편집이 가능하다.

- Data ID Popup을 닫아야만 Text 이외의 다른 편집이 가능하다.

- Data ID Popup을 닫지 않고 다른 Screen을 선택하여 편집하진 못한다.

 ( Popup창을 닫아야만 다른 Screen선택 가능 )

- Data ID Popup창의 크기는 10X10( Row X Column )의 크기를 벗어나는 경우 스크롤이 생긴다.

- Column의 수가 10개 이하일 경우 아래와 같이 Column의 크기에 Fit되어 Width가 결정되어 제공된다.

- Row의 수가 10개 이하일 경우 Row의 크기에 Fit되어 Height가 결정되어 제공된다.

검색된 데이터가 없는 경우

- 검색된 Data ID가 없을 경우 검색결과가 없다는 메세지를 Data List에 띄워준다.

- 메세지는 사용자의 다른 행위를 할 경우 사라진다.

- Data ID검색결과가 List에 나오지 않을뿐 다른 기능들은 정상적으로 동작된다.

- 검색한 검색어는 사용자가 지우지 않는 이상 유지된다.

- 화면상에 보이지 않는 데이터를 찾는 경우 스크롤링이 되어 해당 Data ID값이 선택된다.
