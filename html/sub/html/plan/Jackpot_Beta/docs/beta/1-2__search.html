- 검색창 클릭하면 커서가 깜박 거려서 바로 검색어를 입력할 수 있는 상태가 된다.

- 검색어를 입력하는 순간 검색이 실행된다.

- 검색 결과는 검색어를 입력하는 순간 마다 반영이 된다.

- 검색어는 대소문자를 구분하지 않는다.

- 검색어가 포함된 모든 리스트가 검색 결과로 나타난다.

- 검색어가 입력된 상태에서는 검색어에 대한 결과만 리스트에 나타나게 된다.

- 그룹(폴더)는 검색결과에 포함되지 않는다.

- 사용자가 검색어 삭제를 하지 않으면 어떤한 상황이라도 검색 결과 내용은 유지된다.

- 검색어를 입력하면 검색창 우측에 Delete 버튼이 나타난다.

- Delete 버튼을 누르면 검색어가 지워진다.
