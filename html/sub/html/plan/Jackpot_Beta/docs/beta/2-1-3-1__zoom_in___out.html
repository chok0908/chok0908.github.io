2-1-3-1. Zoom In / Out

Map을 확대하기

- View > Zoom In을 클릭하면 현재 Zoom Level에서 +1 증가한다.

- Screen을 중심을 기준으로 확대된다.

- Object 위치는 변경되지 않는다.

Map을 축소하기

- View > Zoom Out을 클릭하면 현재 Zoom Level에서 -1 감소한다.

- Screen을 중심을 기준으로 축소된다.

- Object 위치는 변경되지 않는다.

선택된 Object가 있은 경우 Map 확대/축소

- Screen 중심이 선택된 Object의 Gripper 중심으로 이동되면서 확대/축소 된다.

- 선택된 Object가 위치 이동하는 것이 아닌 Screen 화면이 이동하는 것이다.
