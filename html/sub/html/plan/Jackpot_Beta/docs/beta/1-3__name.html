이름 입력하기

- 최초에는 'Name'로 자동생성 된다.

- 최초 생성 이후에는 'Name + 빈칸 + 자동생성 숫자'로 자동생성 된다.

- 자동생성 숫자는 1부터 시작한다.

- 중복 이름 허용한다.

- 최초에는 '복사 대상 이름 + _ + Copy'로 자동생성 된다.

- 최초 복사한 이후 동일한 것을 또 복사한 경우 '복사 대상 이름 + _ + Copy + 빈칸 + 자동생성 숫자'로 자동생성 된다.

- 자동생성 숫자는 1부터 시작한다.

- 문자는 특수 문자  # /  : * " < > | 등은 제외하고 모두 허용한다.

- 스페이스(여백)은 앞뒤에 문자가 있어야 허용한다.

ex)a b
