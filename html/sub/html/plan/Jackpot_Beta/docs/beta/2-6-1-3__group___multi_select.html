< Search Bar >

 - Properties 항목을 검색할 수 있다.




< Group Name >

 - Group 이름을 설정할 수 있다. ( 사용자가 정의 )

 - Default값은 없다.

 - Group의 Name을 지정해 주지 않으면 "Group"이라고 Group Name란에 워터마크 처리되어 나온다.

 - Multi Select의 경우 Group Name란이 제공되지 않는다.




< 선택된 대상의 속성 >

 - 선택되어진 객체의 속성을 표시해준다.

 

< Group Object 보기 설정 >

 - Object Map위에서 보여지는 것에 대한 것을 설정한다.

 - None / Lock to Screen / Keep Size 세가지 기능 중 한가지만 선택 가능하다. ( 중복사용 불가 )

 - None : Show상태를 말한다. 화면에서 아무런 기능을 하지 않는다. ( Default값 - None )

 - Lock to Screen : Screen에서 Size를 유지하며 Zoom, Panning 잠금기능이다. ( Group의 좌측 상단 기준 )

 - Keep Size : Map의 Zoom Level과 상관없이 항상 같은 사이즈로 보여지게 설정하는 기능이다.

 - Hide : Map상에 Object를 숨기는 기능이다.( 체크박스제공 )

 - Hide는 상단의 기능과 상관없이 중복 사용가능하다.


< Group Style 설정 >

 - Group된 Object들의 Object Type 속성에 따라 공통속성들만 나온다.








< Text 설정 >

 - Object의 Text속성을 설정할 수 있다.




< Position 설정 >

 - Object의 위치값을 설정할 수 있다.

 - GIS 좌표계를 가지고 있다.

 - X : 위도 좌표 ( Data 입력 값 - 38.5525525 ( 숫자 ) 형식으로 / 소수점 7자리까지 입력 )

 - Y : 경도 좌표 ( Data 입력 값 - 38.5525525 ( 숫자 ) 형식으로 / 소수점 7자리까지 입력 )
