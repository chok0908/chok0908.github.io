2. Ungroup

- Group을 선택하고 Ungroup명령을 실행하면 Group이 해제된다 .

 ( Group이 사라지고 안에 있던 Object가 List에 뿌려지게 된다. )

- Ungroup명령은 Group을 선택했을 때만 실행이 가능하다.

 ( Object만을 선택했을 때는 비활성화 된다. )

- Ungroup명령으로 해제된 Object들은 Select된 상태로 List에 보여진다.

- Multi Select시 Group이 포함되어 있는 Multi Select라면 Ungroup 실행이 가능하다.
