Tag 삭제

 - Tag 삭제는 Tag Icon에서  X버튼을 눌러 삭제한다.

 - Tag의 개수가 많아지면 Text Input Box의 세로 크기가 늘어난다.

   (최대 4줄까지 /  4줄이 넘어가면 스크롤이 생긴다.)

 - Tag의 개수는 최대 10개로 제한한다.

- Tag Name을 입력하면 Tag로 입력된다. ( Tag Name이 Icon으로 생성됨 )

- 추가적인 Tag를 입력할 경우 Name ,(콤마)를 입력하면 새로운 Tag가 입력된다.

  (블로그 글쓰기 Tag 입력과 유사하게 동작)

- Tag Name의 입력 길이에 최대 100자이다.

- 최소입력값은 1자이다.

- 입력값이 없을 경우 자동 취소된다.

3. Tag의 이름이 동일한 경우

It is already a registered name.

Please enter another name.

- 이미 등록된 Tag의 이름을 입력한 경우 Popup창이 띄워진다.

- Popup창엔 이미 등록된 이름입니다. 다른 이름을 넣어주세요. 라는 문구의 Popup창이 출력된다.

- Popup창은 2초 후에 자동적으로 닫힌다.

- 사용자가 Tag이름을 등록하기전 상태로 돌아간다.( 입력요청된 Tag이름은 등록되지 않는다. )
