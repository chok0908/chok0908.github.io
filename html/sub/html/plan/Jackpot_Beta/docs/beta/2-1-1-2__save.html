- Save를 클릭하면 현재 선택된 Screen의 Map을 서버에 저장한다.

- 변경된 내용이 있는 Map들은 Screen Tab의 Title 앞에 "*"가 나타난다.

- 변경된 내용이 있는 Map을 닫을 경우 Save 여부를 확인하는 팝업창이 나온다.

- 저장 버튼을 클릭하면 저장하고 Map이 닫히고 취소 버튼을 클릭하면 저장 안하고 Map이 닫힌다.

- 저장한 시점에 Screen의 중심 위치에 있는 위도, 경도와 Zoom Level을 저장한다.
