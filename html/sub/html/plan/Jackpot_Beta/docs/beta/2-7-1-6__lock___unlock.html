2-7-1-6. Lock & Unlock

1. Lock

Unlock

Lock

- 오브젝트를 생성함과 동시에 Default 속성은 Unlock 상태이다.

- Unlock상태에서는 Object편집이 가능하다.

- Lock상태에서는 모든 편집이 불가능하다. ( Position 변경도 불가 / Properties Position - 비활성화 )

- Unlcok Icon을 클릭하면 Lock Icon으로 변한다. ( Lock 상태 )

- Icon은 토글 버튼이다.

- Selection 되지 않은 Object의 Lock & Unlock의 설정 값을 바꿀 수 있다.

Unlock

Lock

< Icon >

 - Unlock되어 있는 Object는 Screen에서 선택이 가능하고 자유롭게 편집이 가능하다.

 - Lock이 걸린 Object는 Screen에서 선택이 불가능하고 Object편집이 불가능하다.

 - Objects 패널에서도 선택 불가능하다.

 - Object가 Lock된 상태에서도 Show & Hide기능은 동작한다.

 - Lock이 되는 순간 Objects Panel에서 선택해제된다. ( Objects Panel에서도 선택 불가 )

2. Group Lock

- Group을 Lock하게 되면 Group의 하위 Object들은 모두 Lock이 된다.

( 각각의 Object의 Lock여부와 관계없이 모두 적용된다. )

- Group안에 있는 Object를 Lock 시킬 경우 Group편집은 되지 않는다.

- Group안에 있는 Object중에 Unlock 상태인 Object는 편집이 가능하다.
