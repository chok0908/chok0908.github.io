< 서로 다른 설정정보를 가진 경우 >


- 공통속성 아닌 경우 Group에서 보여주지 않는다.

- 설정 항목의 값이 동일 하지 않은 경우 "-"하여 보여준다.

- Multi Select 상황에서도 Size설정을 제외한 모든 설정들은 같은 결과가 출력된다.

- 공통 속성값을 변경하면 Group내의 Object들은 입력된 값으로 변경된다.

 서로 다른 Type일 경우 "-"표시 된다. 같은 Type일 경우 동일한Type이 표시 된다.

Group에 속한 Object들의 전체 크기가 표출된다.

Object들 간의 속성이 같은 경우 같은 속성값이 표출 되지만 다른경우  "-"표시 된다.

Group의 공통 속성이 아닌 항목은 표출되지 않는다.

< 예외 사항/ Multi Select >

- Object들이 서로 다른 크기의 값을 가지고 있을 경우 Size값이 "-"처리 된다.

- Object들이 서로 같은 크기의 값을 가지고 있을 경우 Size값은 같은 크기의 값이 표출된다.

- Fit기능을 사용하게 되면 Multi Select된 Object들은 모두 화면에 Fit 된다.
