Context Menu 숨기기

- Context Menu가 지원하는 영역에서 마우스 우클릭을 하면 팝업 나온다.

- Context Menu가 지원하는 영역이 아닌 곳에서 마우스 우클릭을 하면 아무런 동작 하지 않는다.

- Context Menu가 보이는 상태에서 Context Menu 이외 영역을 마우스 클릭 하면 자동으로 Context Menu는 숨겨진다.

Panel별 Context Menu

- Main Menu / Toolbar / Properties / Libraries는 Context Menu를 지원하지 않는다.

- Maps / Objects / Screen 영역에서만 Context Menu를 제공한다.
